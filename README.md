src/App.vueを編集する。
cssはsrc/styles.scssを見ているが、App.vueに書いても問題ない。

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm build
```

Running

```sh
npm dev
```
